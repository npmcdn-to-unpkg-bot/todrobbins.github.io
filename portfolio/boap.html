Book of Abraham Project

What I Did

  • Designed a new user interface for a collection of religious texts
  • Implemented a basic MySQL database

What's Next

  • Learn how to extract name entities for text retrieval (people, places, and things)
  • Learn how to encode/display the texts with LaTeX

This project has been a collaboration with a Brigham Young University professor over the past few years. He has assembled a large array of resources concerning the life of the Mormon prophet Joseph Smith, Jr., and his successors. I have been assisting with information architecture and designing a new site for him in my free time. I have utilized technologies such as: HTML5, CSS, Javascript, and MySQL/PHP for text retrieval. We also run informal user tests to ensure the user experience is acceptable to casual users and researchers alike.

I have learned many useful skills in the process, including processing large text data sets and manipulating those texts for display on the Web. Our collaboration is ongoing and we intend to release a new version of the site in the coming months.
